师资团队

教职工610人，其中专任教师464人，高级职称114人，博士40人。

实习情况

定期与行业协会、商会联手,每次均有超过400家企业参加，提供近万个就业岗位。不定期组织各类小型专场招聘会。

校外基地

华勤通讯技术有限公司

华勤通讯技术有限公司创建于2005年，是全球领先的多品类智能通讯终端研发设计公司（智能通讯终端ODM），产品涵盖智能手机，平板电脑，笔记本电脑，服务器及IoT产品。总部位于中国上海，并在西安，无锡，东莞，南昌，深圳，台湾，日本，香港，美国，印度设有研发中心，制造基地，运输中心和办事处。截止2017年，华勤拥有员工15000余人，其中职员4000余人（研发人员占65%)，拥有近2000项授权专利。

深圳市金立通信技术有限公司

金立通信技术力量雄厚且对技术的掌握有着长远的规划，是目前国内手机行业设计与研发最深最广的品牌之一。金立通信拥有两家独资加工工厂东莞市金众电子和东莞市金铭电子。总投资1.8亿元，拥有厂房面积60000多平方米，员工3000人，未来计划再投资7000万元，最终建成一个年产能800万部的高度自动化现代化手机专业生产基地。

广东中科遥感技术有限公司

广东中科遥感技术有限公司，挂靠在“中国科学院云计算产业技术创新与育成中心”。该中心由广东省与中国科学院联合建立，是直属于中科院领导的大型院地合作机构。作为我国第一个“遥感云服务研究中心”，广东中科遥感将基于中科院云计算资源，创新遥感云服务技术，开创遥感产业化的新模式。

东莞市得心软件有限公司

公司提供具有竞争力和公平性的薪酬福利；提供多层次的员工培养和职业规划方案；经常组织丰富多彩的员工活动等。持续发展的得心软件欢迎您，希望您在众多的竞争者中脱颖而出，成功加入得心软件，我们真诚的邀请优秀人才加盟我们的团队，共同创业、共同发展。

东莞市百达连新电子商务有限公司

东莞市百达连新电子商务有限公司，是一家以电商团队与人才的培训和输出为核心，以电商平台运营外包服务为基础的电商服务公司，致力于为电商企业量身打造各电商平台高能效运营团队与岗位人才。
